## 📋 One-Click Deployment Steps

### Step 1: Install Adapter (1 minute)
```bash
npm install -D @astrojs/netlify
```

### Step 2: Update astro.config.mjs (1 minute)
```javascript
import netlify from '@astrojs/netlify';
export default defineConfig({
  // ... existing config
  adapter: netlify(),
});
```

### Step 3: Deploy (1 minute)
```bash
npm install -g netlify-cli
netlify deploy --prod
```

### Step 4: Set Environment Variable (1 minute)
In Netlify Dashboard:
```
BITRIX24_WEBHOOK = https://vezemcip.bitrix24.ru/rest/1/cwill2a1sdpyntll/crm.lead.add
```

### Step 5: Verify (5 minutes)
- [ ] Test website loads
- [ ] Add items to cart
- [ ] Submit order form
- [ ] Verify order in Bitrix24
- [ ] Check SEO files accessible

**Total Time: 10 minutes**
